Overview Visão Geral

This application was developed to facilitate the search for articles for research and the development of theoretical references. It will use both the Scopus Search API and the Scopus Abstract Retrieval API to query the Scopus cluster. A CSV file with the article information will be returned if an article is found after the search. Esta aplicação foi desenvolvida para facilitar a busca de artigos para pesquisa e desenvolvimento de referenciais teóricos. Ela usará ambos a Scopus Search API e a Scopus Abstract Retrieval API para consultar o cluster Scopus. Um arquivo CSV com as informações dos artigos será retornado se um artigo for encontrado após a busca.

Keywords Palavras-chave

Based on the theme or subject of your research, you must select a minimum of two, and a maximum of four keywords, which will be used as parameters and filters when searching for articles. Each keyword must be written in English, containing only letters, numbers, spaces and underscores, with a minimum of 2 and a maximum of 50 characters. Com base no tema ou assunto de sua pesquisa, você deve selecionar no mínimo duas e no máximo quatro palavras-chave, que serão utilizadas como parâmetros e filtros na busca por artigos. Cada Palavra-chave deve ser escrita em Inglês, contendo apenas letras, números, espaços e sublinhados, com no mínimo 2 e máximo de 50 caracteres.

API Key Chave da API

You must obtain one to access the Scopus APIs to search and retrieve the articles' information. It has no spaces and is made up of 32 characters containing only letters and numbers. It can be obtained by accessing the Elsevier Portal and registering. Você deve obter uma para acessar as Scopus APIs para pesquisar e recuperar as informações dos artigos. Ela não possui espaços e é composta de 32 caracteres contendo apenas letras e números. Pode ser obtida acessando o Portal Elsevier e registrando-se.
