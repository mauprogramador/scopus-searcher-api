Visão Geral Overview

Esta API foi desenvolvida para facilitar a busca de artigos para pesquisa e desenvolvimento de referenciais teóricos, utilizando a Scopus Search API para consultar a base de dados da Scopus. Para realizar a busca, insira as devidas informações, caso algum artigo seja encontrado, será retornado um arquivo CSV com todas as informações. This API was developed to facilitate the search for articles for research and the development of theoretical references, using the Scopus Search API to query the Scopus database. To perform the search, enter the appropriate information, if an article is found, a CSV file with all the information will be returned.

Palavras-chave Keywords

Based on the theme or subject of your research, you must select a minimum of two and a maximum of four keywords, which will be used as parameters and filters when searching for articles in the API. Each keyword must be written in english, containing only letters, numbers, spaces and underscores, with a minimum of 2 and a maximum of 50 characters. Com base no tema ou assunto de sua pesquisa, você deve selecionar no mínimo duas e no máximo quatro palavras-chave, que serão utilizadas como parâmetros e filtros na busca de artigos na API. Cada Palavra-chave deve ser escrita em inglês, contendo apenas letras, números, espaços e sublinhados, com mínimo de 2 e máximo de 50 caracteres.

API Key Chave da API

It is required to access the API, it can be obtained by accessing the Elsevier Portal. It has no spaces and is made up of 32 characters containing only letters and numbers. É necessária para acessar a API, pode ser obtida acessando o Portal Elsevier. Não possui espaços e é composta por 32 caracteres contendo apenas letras e números.
